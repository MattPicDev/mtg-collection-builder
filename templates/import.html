CSV Format

Your CSV file should have the following columns:

  • Name - Card name
  • Set - Set code (e.g., "NEO", "MID")
  • Collector Number - Card number
  • Quantity - Number owned
  • Foil - "Yes" or "No"
  • Condition - Card condition
  • Language - Card language
Example:
Name,Set,Collector Number,Quantity,Foil,Condition,Language
Lightning Bolt,NEO,123,4,No,Near Mint,English
Counterspell,MID,456,2,Yes,Near Mint,English
Important Notes
  • The tool will look up each card using the Scryfall API to verify details
  • Cards that cannot be found will be skipped
  • Import may take some time for large collections
  • Duplicate entries will overwrite existing ones
